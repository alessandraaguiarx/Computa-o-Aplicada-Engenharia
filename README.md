# Computação Aplicada à Engenharia
Rede de Petri aplicada à gestão de entulho na construção civil. O repositório apresenta a simulação de uma Rede de Petri simples no HPSim, uma Rede de Petri Colorida no CPN IDE, a modelagem do robô Otto adaptado à construção civil feita no Fusion 360 e a programação do processo em C++.
